Cloud Infrastructure

Currently I am working in a Cloud Infrastructure team, where we build from scratch infrastructure for multi-tenant services in AWS. This project requires all our team members to develop high quality, scalable, secure, automated infrastructure that will serve the multi-tenancy purposes. I've implemented multiple key features, worked on networking from API Gateway to the Kuberentes Cluster, and on routing within the cluster. On top of that I have also created a pipeline (using AWS Code Build and AWS Code Pipeline), to automate deployment of our infrastructure whenever a merge, to certain branches is made.

While working on this project I have been honoured with receiving "Team Player" award for my continuous commitment to not only the product, but also team development.

Some of the AWS services that I've worked on are: API Gateway, EKS, ALB, NLB, VPC, Cognito, DynamoDB, Lambda, Code Build, Code Pipeline, CloudFormation
