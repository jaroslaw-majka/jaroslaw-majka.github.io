Blog O mnie Projekty

Dla firmy Adtran miałem przyjemność uczestniczyć w budowie Mosaic API. Aplikacji, która łączyła wiele skomplikowanych komponentów oraz informacji z zewnętrznych API dając klientom możliwość łatwej integracji ze swoimi systemami.

W zespole byłem odpowiedzialny za tworzenie oprogramowania, mentoring młodszych członków zespołu, planowanie i przygotowanie nowych funkcjonalności, automatyzację procesów releasowych, oraz testowanie.

W projekcie bazowaliśmy głównie na Pythonie z AWS Lambda, API Gateway, CloudFormations, S3, DynamoDB, SNS, SQS, oraz DataDog.

Technologie: AWS Python

Przez około rok pracowałem w zespole Cloud Infrastructure, gdzie od podstaw budowaliśmy infrastrukturę dla usług multi-tenantowych w AWS. Projekt ten wymagał od każdego członka zespołu tworzenia skalowalnej, bezpiecznej, zautomatyzowanej infrastruktury wysokiej jakości. Odpowiadałem za wiele kluczowych funkcjonalności, pracując nad siecią — od API Gateway aż po klastry Kubernetes, a także nad routingiem wewnątrz samego klastra.

Dodatkowo stworzyłem pipeline (z wykorzystaniem AWS CodeBuild i AWS CodePipeline), który automatyzuje wdrażanie infrastruktury po mergu do odpowiednich branchy.

Za swoją pracę nad tym projektem otrzymałem wyróżnienie „Team Player” — za nieustanne zaangażowanie nie tylko w rozwój produktu, ale również samego zespołu.

Technologie: Kubernetes AWS Python Java JavaScript

Technologie: JavaScript React

Repozytorium: MicroStrategy/mstrio-py

Technologie: Python Pytest

Aplikacja, którą stworzyłem głównie dla siebie, by mieć kontrolę nad książkami, które przeczytałem, chcę przeczytać lub pożyczyłem komuś (albo od kogoś).
